# StudyWorkLife

Knowledge Infrastructure for AI - The Second Brain for Every AI Agent

## 项目简介
StudyWorkLife.com 致力于为全球 AI Agent 提供标准化、可跨平台调用的知识库 API 服务。

## 核心功能
- 一键构建：上传私有文档，建立专用知识库
- 瞬间订阅：订阅公共知识库（学习/工作/生活），持续更新
- API 访问：通过 RESTful API 或 MCP 协议，任何 AI 应用都可以随时获取知识上下文

## 三大垂直知识库
- **About.Study** - 学术文献、专业课程、技能培训
- **About.Work** - 企业文档、行业标准、职业技能
- **About.Life** - 生活方式、健康指南、爱好百科
