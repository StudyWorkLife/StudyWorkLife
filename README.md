## Core Product: KBaaS

### Universal Knowledge Base Service for AI Agents

**One-Click Build**
Upload private documents and build a dedicated knowledge base. Get production-grade retrieval without complex setup.

**一键构建**
上传私有文档，建立专用知识库。无需复杂设置，即可获得生产级检索能力。

**Instant Subscribe**
Subscribe to public knowledge bases covering Study, Work, and Life with continuous updates.

**瞬间订阅**
订阅公共知识库（学习/工作/生活），持续更新，即时可用。

**API Access**
Through RESTful API or MCP protocol, any AI app can pull knowledge context anytime.

**API 访问**
通过 RESTful API 或 MCP 协议，任何 AI 应用都可以随时获取知识上下文。
