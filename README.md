# StudyWorkLife 知识基础设施

## Project Overview / 项目简介
**English:** Knowledge Infrastructure for AI - The Second Brain for Every AI Agent
**中文:** AI时代的知识基础设施，为全球AI Agent提供标准化、可跨平台调用的知识库API服务

---

## About / 关于
**English:** StudyWorkLife.com is dedicated to providing standardized, cross-platform knowledge base API services for AI Agents worldwide.

**中文:** StudyWorkLife.com 致力于为全球 AI Agent 提供标准化、可跨平台调用的知识库 API 服务。

---

## Core Features / 核心功能

### One-Click Build / 一键构建
**English:** Upload private documents and build a dedicated knowledge base. Get production-grade retrieval without complex setup.

**中文:** 上传私有文档，建立专用知识库。无需复杂设置，即可获得生产级检索能力。

### Instant Subscribe / 瞬间订阅
**English:** Subscribe to public knowledge bases covering Study, Work, and Life with continuous updates.

**中文:** 订阅公共知识库（学习/工作/生活），持续更新，即时可用。

### API Access / API访问
**English:** Through RESTful API or MCP protocol, any AI app can pull knowledge context anytime.

**中文:** 通过 RESTful API 或 MCP 协议，任何 AI 应用都可以随时获取知识上下文。
